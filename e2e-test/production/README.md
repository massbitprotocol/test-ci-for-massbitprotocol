## How to run 
```shell
robot --variable  SSH_HOST:[ADD_HOST_HERE] --variable SSH_PORT:[ADD_PORT_HERE] --variable SSH_USERNAME:[ADD_USERNAME_HERE] --variable SSH_PASSWORD:[ADD_PASSWORD_HERE] production.robot 


```
Example
```shell
robot --variable  SSH_HOST:6.tcp.ngrok.io --variable SSH_PORT:18268 --variable SSH_USERNAME:massbit --variable SSH_PASSWORD:[ADD_PASSWORD_HERE] production.robot
```
